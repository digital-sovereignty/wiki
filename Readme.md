# Digital Sovereignty Wiki
We use [Material for MKDocs](https://squidfunk.github.io/mkdocs-material/).

# Getting started
1. Clone the repository
2. Make sure you have a current Python version (e.g. 3.12) installed.
3. Create and activate a new virtual environment.
4. Install the requirements with `pip install -r requirements.txt`
5. Run `mkdocs serve --livereload` for an auto-updating preview in your browser.
6. Start writing! Please mind the supported plugins listed in `mkdocs.yml`. Feel free to add new extensions.

# Contribution / Deployment
Once your pull-request is accepted, an automated job will re-deploy the site on [wiki.digital-sovereignty.help](wiki.digital-sovereignty.help). There are no manual steps you need to take.

Please provide a helpful description for our reviewers in your pull request.
